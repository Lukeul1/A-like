A-like Word Game

Guess the Word:

Lives: 3

How to Play:

  • Guess the word by entering one letter at a time in the input fields or using the on-screen keyboard.
  • A synonym has been provided for each word.
  • Click the "Check" button to check if your guess is correct.
  • You have 3 lives. Each incorrect guess will deduct 1 life.
  • If you run out of lives, the game is over.
